Logiciels et accessoires recommandés pour mettre le système en pratique, classés par système d'exploitation. Cette page accompagne le guide et est mise à jour à mesure que les références produits évoluent.

Principe du guide : natif d'abord, logiciel libre ensuite, propriétaire seulement en dernier recours. Les leviers natifs de Windows 11, macOS et Linux Mint sont détaillés dans le livre ; cette page recense seulement ce qui demande une installation ou un achat séparé.

ls windows-11/

PowerToys — FancyZones

Suite d'utilitaires officielle Microsoft, gratuite et open-source. Le module FancyZones découpe l'écran en zones fixes pour ranger les fenêtres au glisser-déposer ou au clavier — utile pour qui travaille avec plusieurs fenêtres ouvertes en permanence et veut éviter la recherche visuelle. Installation directe, sans compte requis.

gratuit, open-source (MIT) · voir le site officiel →

Cold Turkey Blocker

Bloqueur de sites et d'applications au niveau du système, pas seulement du navigateur. Une fois un blocage activé, impossible de le contourner avant l'échéance fixée — y compris en redémarrant ou en désinstallant le logiciel. Plus radical qu'une extension de navigateur, pertinent si la distraction se déplace d'un site bloqué à un autre. Identique sur macOS.

gratuit en version limitée, version Pro 39 € en paiement unique (pas d'abonnement) · Windows et macOS, pas de version Linux · voir le site officiel →

ls macos/

Rectangle

Équivalent de FancyZones pour macOS : raccourcis clavier et zones d'accrochage pour ranger les fenêtres sans dégainer la souris à chaque réorganisation. Gratuit et open-source, une version Pro payante existe pour qui veut aller plus loin (gestes, mise en page d'applications), mais la version de base suffit pour l'usage décrit dans le guide.

gratuit, open-source (version Pro optionnelle) · voir le site officiel →

Cold Turkey Blocker (voir section Windows 11 ci-dessus) fonctionne à l'identique sur macOS.

ls linux-mint/

Gestion des fenêtres native (Cinnamon)

Linux Mint propose nativement l'essentiel de ce que FancyZones et Rectangle ajoutent ailleurs : raccourcis de rangement de fenêtres et bureaux virtuels configurables dans Paramètres système → Fenêtres et Effets. Aucune installation nécessaire — c'est le levier natif détaillé dans le guide, mentionné ici pour mémoire.

natif, gratuit · déjà sur votre installation Linux Mint

DigitalZen

Bloqueur de sites et d'applications au niveau du système, équivalent fonctionnel de Cold Turkey mais disponible aussi sur Linux (X11 et GNOME sous Wayland) — rare chez les outils de ce type. Fonctionne aussi sur Windows et macOS, ce qui en fait une option pour qui veut un seul outil sur plusieurs machines avec des systèmes différents.

gratuit en version limitée, Premium ~3-5 $/mois (≈2,80-4,60 €) ou formule à vie ~120 $ (≈111 €) · Windows, macOS et Linux · voir le site officiel →

ls navigateur-et-notes/

Firefox

Le navigateur de référence du guide : open-source, indépendant des grands écosystèmes publicitaires, disponible à l'identique sur Windows, macOS et Linux. Les réglages de réduction du bruit visuel décrits dans le livre s'appuient sur ce navigateur.

gratuit, open-source · voir le site officiel →

uBlock Origin

Extension de blocage de publicités et traqueurs, légère et open-source. Réduit une bonne partie du bruit visuel d'une page avant même d'agir sur les réglages du navigateur lui-même. Disponible pour Firefox sur les trois systèmes.

gratuit, open-source (GPL-3.0) · installer pour Firefox → · code source sur GitHub →

Obsidian

Prise de notes locale en fichiers texte Markdown — aucun stockage cloud imposé, aucun compte requis pour l'usage de base. Utile pour qui veut une version numérique de la logique de capture et de relecture déjà présente dans le système à trois carnets, sans dépendre d'un service propriétaire fermé. Le logiciel lui-même n'est pas open-source, mais le format de fichier (Markdown) l'est, ce qui garantit de pouvoir toujours récupérer ses notes ailleurs. La synchronisation entre appareils (Obsidian Sync) est un service optionnel payant ; elle n'est pas nécessaire pour un usage sur une seule machine.

application gratuite pour usage personnel et professionnel · Sync optionnel ~4 $/mois (≈3,70 €) · voir le site officiel →

Souris sans fil au clic volontairement silencieux, Bluetooth ou récepteur USB 2,4 GHz. Format compact et léger, pensé pour le transport. Compatible Windows, macOS, Linux, iPadOS et Chromebook.

~35 € · voir sur Amazon →

Trust Puck

Souris plate et silencieuse, rechargeable, Bluetooth ou récepteur USB 2,4 GHz, bouton de défilement précis. Alternative économique au modèle Logitech ci-dessus, pour le même usage. Compatible Windows, macOS, Linux, iPadOS et Chromebook.

~17 € · voir sur Amazon →

Trust Vaiya

Clavier sans fil Bluetooth et 2,4 GHz, disposition AZERTY française avec pavé numérique, rechargeable, 55 % de plastique recyclé. Alternative économique au modèle Logitech ci-dessus. Compatible Windows, macOS, iOS, iPadOS et Android.

~35 € · voir sur Amazon →
