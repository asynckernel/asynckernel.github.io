Trois carnets
pour reprendre pied.

Liste des outils recommandés pour mettre le système en pratique. Cette page accompagne le guide et est mise à jour à mesure que les références produits évoluent.

Les liens d'achat sont en cours de finalisation. Les descriptions ci-dessous restent valables pour orienter votre choix.

ls carnets-pointillés/

Leuchtturm1917 — A5 pointillé

La référence des carnets bullet journal. Pages numérotées, index pré-imprimé, deux marque-pages, pochette à l'arrière. Le choix par défaut sans réflexion. Encre fluide qui peut traverser légèrement le papier — à éviter pour feutres et pinceaux.

~22-25 € · voir sur Amazon →

Rhodia Goalbook — A5

Alternative française de qualité supérieure. Papier plus épais et plus blanc, confort d'écriture pour stylos plume et feutres. Index pré-imprimé, table des matières détaillée.

~22 € · voir sur Amazon →

Muji — Carnet pointillé A5

Entrée de gamme propre pour démarrer sans engagement. Couverture simple, papier honnête, prix imbattable. Idéal pour confirmer si la pratique vous convient avant d'investir dans un modèle durable.

~5-8 € · voir sur le site Muji →

ls agendas-multi-niveaux/

Brunnen — Agenda annuel A5

Modèle multi-niveaux classique avec vue annuelle, mensuelle et hebdomadaire à créneaux horaires. Reliure spirale, format pratique. Multilingue (DE/EN/FR/IT). Le modèle utilisé pour le système décrit dans le guide.

~18-25 € · voir sur Amazon →

Quo Vadis — Trinote A5

Référence française. Vue hebdomadaire avec créneaux horaires, vue mensuelle, vue annuelle. Reliure cousue solide, disponible en plusieurs millésimes.

~15-20 € · voir sur Amazon →

Moleskine — Agenda 18 mois

Pour démarrer en milieu d'année. Format compact, couverture rigide, présentation soignée. La pagination 18 mois couvre une année scolaire complète.

~22-28 € · voir sur Amazon →

ls petits-carnets/

Field Notes — Pack de 3

Carnets souples format poche agrafés. Conçus pour la capture rapide, sans souci de pérennité. Idéal pour le brain dump nocturne ou la capture en mobilité.

~12-15 € le pack · voir sur Amazon →

Moleskine — Cahier A6 (pack de 3)

Format A6 souple, couverture cartonnée robuste, papier crème confortable. Bonne tenue dans le temps malgré la souplesse. Idéal pour la table de nuit.

~12-18 € · voir sur Amazon →

ls stylos/

Pilot G2 — 0.5 ou 0.7 mm

La référence universelle. Écriture fluide, encre tenace, prise confortable. Disponible en plusieurs largeurs de pointe. Mécanisme à clic un peu sonore, à éviter en situation calme.

~3 € · voir sur Amazon →

Uni-ball Signo — 0.38 mm

Trait fin idéal pour les petites écritures. Encre gel précise, sans bavure. Adapté aux journaux et carnets compacts où chaque millimètre compte.

~4 € · voir sur Amazon →

Lamy Safari — stylo plume

Stylo plume entrée de gamme remarquable. Prise ergonomique anatomique, mécanisme robuste. Sensation d'écriture incomparable une fois adopté. Cartouches ou convertisseur.

~25-30 € · voir sur Amazon →

ls feutres-tracker-mensuel/

Stabilo Boss — 5 couleurs

Cinq surligneurs pour le tracker mensuel d'énergie (rouge, orange, jaune, vert clair, vert foncé). Cohérence visuelle des couleurs Stabilo, encre généreuse.

~6-10 € · voir sur Amazon →

Stabilo Pen 68 — feutres fins

Alternative aux surligneurs si vous préférez tracer des aplats nets dans les cases du tracker. Pointe fine, encre vive, palette étendue (vous choisissez les 5 nuances).

~8-12 € le set · voir sur Amazon →
