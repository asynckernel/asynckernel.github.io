Comment Microsoft fait pour vous empêcher de choisir Firefox

Un rapport de 2026 documente les tactiques de design de Windows qui court-circuitent le choix du navigateur. Ce que ça change pour les profils neuroatypiques.

22/07/2026

Ce que Microsoft fait pour vous empêcher de choisir Firefox

En juillet 2026, Mozilla a publié la deuxième édition d'un rapport d'investigation indépendant intitulé Over the Edge 2.0. Les auteurs, Harry Brignull et Cennydd Bowles, ont audité le parcours utilisateur sur Windows 10 et Windows 11 dans quatre pays : États-Unis, Royaume-Uni, Inde et Allemagne. Leur conclusion est sans nuance : Microsoft continue de déployer des techniques de design conçues pour vous maintenir dans Edge, quelles que soient vos préférences déclarées.

Ce que le rapport documente

Les chercheurs identifient six catégories de procédés. Formulation trompeuse : les dialogues suggèrent qu'Edge est nécessaire ou supérieur. Obstruction : chaque étape pour installer un autre navigateur ou le définir comme défaut multiplie les clics et les confirmations. Interférence visuelle : Edge est mis en avant, les alternatives reléguées. Présélection : Edge est coché par défaut dans les options. Harcèlement : des pop-ups répétées rappellent l'existence d'Edge après chaque installation concurrente. Action forcée : certaines fonctionnalités Windows — dont Copilot — ouvrent des liens dans Edge même si votre navigateur par défaut est un autre.

Un cas est particulièrement révélateur. Lors d'une migration de Windows 10 vers Windows 11 via l'outil Windows Backup, le navigateur par défaut préalablement configuré n'est pas transféré. Edge reprend silencieusement cette place. La transition de système d'exploitation devient un mécanisme de réinitialisation des choix.

Pourquoi ça compte davantage pour les profils neuroatypiques

Le tome 3 de la collection repose sur un principe simple : chaque interruption non souhaitée consomme une ressource cognitive qui ne se régénère pas instantanément. Les techniques documentées par le rapport Mozilla ne sont pas anodines dans ce contexte.

Edge est construit autour d'un modèle commercial publicitaire. Il intègre des recommandations de contenu, des suggestions d'achat, des notifications promotionnelles et un assistant Copilot. Autant de sources de stimulation parasites que les chapitres 12 à 14 du tome 3 cherchent précisément à neutraliser — en commençant par le choix du navigateur lui-même.

Firefox est différent sur deux points concrets. D'abord, il est développé par une organisation à but non lucratif (la Mozilla Foundation) dont le modèle économique ne repose pas sur la valorisation des données de navigation. Ensuite, son catalogue d'extensions — uBlock Origin au premier rang — permet de désactiver chirurgicalement les publicités, les animations, les suggestions contextuelles et les éléments d'interface distrayants. Le mode lecture natif de Firefox, mentionné au chapitre 14, offre une vue épurée de n'importe quelle page sans installation supplémentaire.

Pour les profils HPI, TSA ou TDA/H, ce n'est pas une question d'idéologie. C'est une question de surface de stimulation. Moins Edge en a, mieux c'est — mais Edge en veut plus, pas moins.

Une donnée souvent sous-estimée : la vie privée comme protection concrète

La navigation sous Edge, combinée aux consentements Windows et à Copilot, constitue selon le rapport un pipeline potentiel de données qui peut inclure la navigation effectuée dans d'autres navigateurs. Ce point mérite attention pour des raisons pratiques qui dépassent le principe.

Les données de navigation peuvent renseigner les assureurs, les organismes de crédit ou tout acteur qui accède à des profils comportementaux agrégés. Pour les personnes neuroatypiques, dont les parcours de santé, les achats ou les recherches peuvent être atypiques, l'exposition de ces données présente un risque réel — difficile à quantifier, mais difficile à ignorer.

Firefox, avec ses réglages par défaut renforcés et l'extension uBlock Origin, réduit mécaniquement cette surface d'exposition. Pas à zéro. Mais de façon mesurable.

Ce que vous pouvez faire

Installer Firefox ne suffit pas. Il faut aussi le définir comme navigateur par défaut — et le maintenir comme tel, malgré les sollicitations de Windows. Le chapitre 12 du tome 3 détaille la procédure pour chacun des trois systèmes couverts (Windows 11, macOS, Linux Mint).

Si vous utilisez Windows 11, deux précautions supplémentaires : désactiver Copilot dans les paramètres Windows, et vérifier que les liens ouverts depuis d'autres applications ne sont pas redirigés vers Edge (paramètre Applications par défaut > Navigateur).
